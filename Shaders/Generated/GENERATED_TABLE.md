# Generated Shader Assets

Generated shader files live here or in child folders. The generator manifest table lives at [`ShaderSystem/GeneratedManifests/GENERATED_TABLE.md`](../../ShaderSystem/GeneratedManifests/GENERATED_TABLE.md).

Rules:

- Do not edit generated shader bodies by hand.
- Generated shaders must include a provenance header.
- Generated shaders must not contain `_lil*`, `_HoAov*`, `HoAOV`, `HoAOVSSS`, or `lilToonOIT`.
